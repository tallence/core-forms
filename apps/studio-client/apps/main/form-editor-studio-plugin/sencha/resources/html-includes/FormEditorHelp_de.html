Über die Formular-Elemente wird Ihr Formular zusammengestellt. Es wird dabei zwischen einzelnen Elementen und Element-Gruppen unterschieden. Element-Gruppen bilden mehrere Formular-Elemente auf einmal ab. Im Folgenden soll die Bedienung des Editors erläutert werden. Für alle Felder kann angebenen werden, ob es sich um ein Pflichtfeld handelt. Sowie ein Hinweistext. Wird das Element vom Nutzer angeklickt, erscheint der Text rechts daneben.

Element Beschreibung
Textfeld

Ein einfaches Textfeld. Zusätzlich kann konfiguriert werden, wie lang die Eingabe maximal sein darf.

Text-Area

Ein erweitertes Textfeld zur Eingabe von längeren Texten. Zusätzlich kann die Anzahl der Reihen (Höhe) der Text-Area angegeben werden.

Datei Upload

Ein Feld zum Hochladen von Dateien. Zusätzlich kann angegeben werden, wie groß die Datei maximal sein darf (in KiloBytes). Wird nichts eingegeben, ist die maximale Dateigröße 3MB (3072KB).

Radio-Buttons

Eine Auswahl einer Option aus mehreren Möglichkeiten. Es können beliebig viele Möglichkeiten (Buttons) angelegt und auch wieder gelöscht werden.

Check-Boxes

Eine Auswahl einer oder mehrerer Optionen aus mehreren Möglichkeiten. Es können beliebig viele Möglichkeiten (Boxen) angelegt und auch wieder gelöscht werden.

Select-Box

Eine Auswahl einer Option aus mehreren Möglichkeiten. Es können beliebig viele Möglichkeiten angelegt und auch wieder gelöscht werden.

Datums-Feld

Ein Textfeld zur einfachen Datumseingabe. Zusätzlich lassen sich auch Datumsgrenzen festlegen.

Nutzer-Mail

Hier werden zwei Felder ausgegeben: Ein Textfeld für die E-Mail-Adresse des Nutzers und eine Check-Box, mit der der Nutzer steuern kann, ob eine Kopie der Formulardaten an ihn per E-Mail gesendet werden soll.

Hinweistext

Hier wird nur Text ausgespielt, beispielsweise eine Hinweistext. Für den Nutzer besteht keine Möglichkeit Daten einzugeben.
