The form is created using the different form elements. There is a distinction between simple elements and element groups (complex elements). Element groups are collections of multiple elements. Below you can find a simple explanation for the usage of the editor. For each element, you can select whether it is mandatory, add a hint which is shown to the right of an element.

Element Description
Text field

A simple text field. You can configure min/max lenghts and add a regular expression which the result has to match.

Text area

An expanded text field for entering larger texts. You can configure its height.

File upload

A field for uploading files with a configurable maximum file size (in KB). If no size is entered, the default is 3 MB (3072 KB).

Radio buttons

A list of options from which one can be selected. You can added and remove an unlimited (within reason) number of items.

Check boxes

A list of options from which one or more can be selected. You can added and remove an unlimited (within reason) number of items.

Select box

A list of options displayed as a drop-down-box from which one can be selected. You can added and remove an unlimited (within reason) number of items.

Date field

A simple field for date input. You can configure a minimum and a maximum date.

User-Mail

Displays two fields: A text field for the users e-mail address and a check box which allows the users to control if they want to receive a copy of the submitted form.

Hint

Not actually a form element as such. Will only display the entered text (as a help or hint for users).
